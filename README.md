# OverSwissBot
Over Swiss Bot for Discord using Node.js

## Installation
* clone the repo
* use: "npm install" to install all dependencies
* start the bot with: "node bot.js"

## Commands
### Overwatch
* -quick <BattleTag>
* -comp <BattleTag>
* -quickraw <BattleTag>
* -compraw <BattleTag>
### RPG
* -startrpg
* -profile
* -daily
### Dev Stuff
* .eval
